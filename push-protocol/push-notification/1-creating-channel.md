# Creating a Channel --takes 5 min.

## Channel

To send notifications to someone on Web3, you need something that's going to send it -- that something is called the channel.

## Analogy

-   Youtube = Push Protocol
-   Channel = Channel
-   Subscriber = Subscriber
-   Subscribe = Opt-In
-   Unsubscribe = Opt-Out
